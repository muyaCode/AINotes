ChatGPT全面工具箱：在任何地方使用 ChatGPT ​

浏览器扩充套件 ​

  • ChatGPT for Google：Chrome/Edge/Firefox 浏览器扩充套件，在 Google 搜索结果旁并列 ChatGPT 回应。（Firefox扩充套件、程式码）
  • ChatGPT Extension：Chrome 浏览器扩充套件，在右上角弹窗快速使用 ChatGPT。（程式码）
  • ChatGPT Everywhere：Chrome 浏览器扩充套件，在任何输入框使用 ChatGPT。（demo）

桌面应用程式 ​

  • ChatGPT Desktop App：Windows/MacOS 桌面选单列应用程式。使用 Tauri 和 Rust 开发。
  • chatgpt-mac：MacOS 选单列应用程式。

编辑器 ​

  • ChatGPT for VSCode：VSCode 扩充套件。（demo）
  • intellij-chatgpt：JetBrains 编辑器扩充套件。
  • chatgpt.vim：Neovim 插件，在 Neovim buffer 裡使用 ChatGPT。
  • docGPT：Google 文件编辑器插件，在 Google Docs 内使用 ChatGPT。

启动器（Launcher）扩充套件 ​

  • ChatGPT Raycast extension：Raycast 扩充套件。

聊天机器人 ​

  • Twitter/@ChatGPTBot：Twitter 推特机器人。（程式码）
  • ChatGPT ProBot：GitHub APP。输入 /chatgpt 来与 ChatGPTBot 交谈。
  • chatgpt-telegram：执行自己的 GPTChat Telegram 机器人，只需要一行指令。
  • ChatGPT Telegram Bot in AWS Lambda: ChatGPT Telegram 机器人，执行在 AWS Lambda 上。支援语音讯息和 Markdown 显示。
  • chatbot-telegram: 一键运行 ChatGPT Telegram Bot，使用 Deno 和 TypeScript 打造。
  • chatGPT-discord-bot：整合自己的 Discord 机器人。
  • wechat-chatgpt：ChatGPT 的微信 Bot。装完依赖后只需要填写 OpenAI 帐号密码和微信扫码就可以使用。
  • ChatGPT LINE Bot（中文内容）：建立自己的 ChatGPT LINE 机器人。
  • gpt-ai-assistant（中文内容）：在 10 分钟内打造自己的 GPT LINE 机器人。
  • chatgpt-for-chatbot-feishu：快速将 ChatGPT 接入飞书的应用，基于 OpenAI 官方接口 GPT3 模型，支持私有部署，作为私人工作助理或者企业员工助理。

扩展 ChatGPT 功能 ​

  • ShareGPT: 轻分享 ChatGPT 对话纪录。（demo）
  • ChatGPT export to PNG/OpenAI-ChatGPT/HTML：Chrome 浏览器扩充套件，能将 ChatGPT 纪录下载成 PNG、PDF 档，或产生可分享的连结。
  • ChatGPT Advanced: Chrome 浏览器扩充套件，在 ChatGPT 页面同时显示网页搜寻结果。
  • TampermonkeyUserscripts/ChatGPTVoiceInput（中文内容）: Tampermonkey 脚本，ChatGPT 语音输入介面（支援中/英/日/韩语言）。

延伸应用 ​

  • summarize.site：Chrome 浏览器扩充套件，总结网页内容、给出摘要。
  • commitgpt：自动产生 commit 讯息。
  • chatgpt-action: Github Action，让 ChatGPT review 你的 PR。
  • StackExplain: 让 ChatGPT 解释错误讯息的命令列工具。
  • ChatMind：透过人工智慧对话生成心智图。

提示词（Prompts） ​

  • showGPT: 蒐集优质提示词的网站，你也可以轻鬆上传更多提示词。
  • awesome-chatgpt: 各种 ChatGPT 提示词用法的文章和想法。
  • Awesome ChatGPT Prompts: ChatGPT 提示词清单。
  • ChatGPT 指令大全（中文内容）: 精炼过的指令语句，让你充分发挥 ChatGPT 的强大功能。

开发工具（API、SDK） ​

  • PyChatGPT（Python）：轻量化、基于 TLS 的 ChatGPT API 可以在您的 CLI 中使用，无需使用浏览器或验证金钥。
  • pip/revChatGPT（Python）：与 OpenAI 的 ChatGPT API 互动的轻量化套件。採用逆向工程手法。
  • npm/chatgpt（Node.js）：Node.js 客户端。

云函数 ​

接下来我们就可以开始动手打造自己的 ChatGPT 了，首先把上一节的函数替换为下面的内容：

typescript
import cloud from '@lafjs/cloud'

export async function main(ctx: FunctionContext) {
  const { ChatGPTAPI } = await import('chatgpt')
  const data = ctx.body

  // 这里需要把 api 对象放入 cloud.shared 不然无法追踪上下文
  let api = cloud.shared.get('api')
  if (!api) {
    api = new ChatGPTAPI({ apiKey: cloud.env.CHAT_GPT_API_KEY })
    cloud.shared.set('api', api)
  }

  let res
  // 这里前端如果传过来 parentMessageId 则代表需要追踪上下文
  if (!data.parentMessageId) {
    res = await api.sendMessage(data.message)
  } else {
    res = await api.sendMessage(data.message, { parentMessageId: data.parentMessageId })
  }
  return res
}
